== Description ==

Thousands of branded products available in real time into your Woocommerce store.

Start selling clothing, bags, watches, sunglasses and many others products of the best fashion brands from brandsdistribution.com directly into your Woocommerce web store.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/brandssync` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->BrandsSync screen to configure the plugin
